# Mastermind Web Game

Projeto desenvolvido como case técnico. Trata-se de uma versão web do clássico jogo Mastermind.

## Stack Tecnológica
- **Backend:** Java 17+, Spring Boot, Maven, PostgreSQL
- **Frontend:** Angular 15+, Node.js

## Estrutura
O projeto está dividido em duas partes principais:
- `/backend`: API RESTful
- `/frontend`: Aplicação Web Angular
